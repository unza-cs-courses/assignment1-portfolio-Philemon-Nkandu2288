About Me

I'm a passionate software developer with a strong foundation in building clean, efficient, and user-centred web applications. My journey into software development started with a curiosity about how the websites and apps I used every day actually worked — that curiosity quickly turned into a career path I'm genuinely excited about. Through my studies in Computer Science, I've gained hands-on experience across the full web development stack — from crafting responsive front-end interfaces with HTML, CSS, and JavaScript, to building back-end logic and working with databases. I enjoy projects that sit at the intersection of functionality and great user experience, where good engineering and thoughtful design work together.I'm especially drawn to JavaScript and its ecosystem, and I'm actively deepening my knowledge of frameworks like React and Node.js. I believe in writing code that is not only functional but readable and maintainable — code that other developers can build on confidently.Outside of coursework, I work on personal projects that challenge me to solve real problems, and I enjoy collaborating with others, picking up new tools quickly, and contributing positively to any team I'm part of. After graduation, I aim to join a team where I can keep growing, ship meaningful products, and make a real impact through technology.

My Projects

Weather Dashboard

A responsive weather web app that fetches real-time weather data and 5-day forecasts for any city worldwide using the OpenWeatherMap API. Users can search by city name, toggle between Celsius and Fahrenheit, and save favourite locations. Built with vanilla JavaScript, HTML5, and CSS3 with a mobile-first design approach.

Live Demo GitHub

Task Manager App

A full-stack productivity application that allows users to create, organise, and track daily tasks. Features include user authentication, drag-and-drop task reordering, priority labels, and deadline reminders. Built with React on the front end and Node.js with Express on the back end, using MongoDB for data persistence.

Live Demo GitHub

Student Grade Tracker

A web-based grade management system designed for students to log their course marks, calculate weighted averages, and visualise academic progress through interactive charts. Built with Python and Flask for the back end, with a SQLite database, and Chart.js on the front end to render grade trend graphs.
